- bộ dataset sử dụng: tấn công ddos trong CIC DoS dataset (2017)(link: https://www.unb.ca/cic/datasets/dos-dataset.html)
- Chạy file ddos_app.py qua cmd với quyền admin
- Các file phải cùng 1 thư mục: 
	+ ddos_ui.py (file chứa giao diện)
	+ ddos_app.py (file chạy)
	+ config.txt  (file cấu hình)
	+ ddos_detect_model.sav	(file chứa model đã train)
Lưu ý: Tắt windows security để có thể chạy tool LOIC
